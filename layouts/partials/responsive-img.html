{{- /* Rewrites micro.blog-hosted JPG in post content to a small/medium srcset. micro.blog can resize .jpg via -s (300px) and -m (~1000px) variants but NOT .webp, so .webp and external images are left untouched (graceful during webp->jpg migration). micro.blog emits self-hosted images as relative paths (src="uploads/...") and some as absolute cdn.uploads.micro.blog URLs; match both. Strip any existing -xl/-m/-s suffix first so abc-xl.jpg rebuilds from the base (abc-s.jpg), not the 404 abc-xl-s.jpg. */ -}} {{- return (. | replaceRE `src="(/?uploads/[^"]*?|https://cdn\.uploads\.micro\.blog/[^"]*?)(?:-xl|-m|-s)?\.(jpe?g)"` `src="${1}-s.${2}" srcset="${1}-s.${2} 300w, ${1}-m.${2} 1000w" sizes="(min-width: 571px) 32vw, 100vw"`) -}}
